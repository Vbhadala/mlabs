# v1 — Current

**Tokens:**
- `primary: oklch(0.69 0.18 39)` (~#FF6B2C, bright orange)
- `primaryForeground: oklch(0.205 0 0)` (~#1F1F1F, near-black)

**Contrast pairs:**
- White-on-primary: **2.6:1** — fails WCAG AA body
- Dark-on-primary: **7.5:1** — passes AA + AAA
- Primary-on-white: **3.4:1** — fails AA body (text-primary in marketing)

**Visual character:**
- Bright, saturated, attention-grabbing "candy orange"
- Reads as fluorescent / playful
- Dark text on the orange CTA looks deliberate but slightly retro / unfinished

**Why this exists:** the only way to satisfy the contrast hook with this
particular bright orange is dark-on-orange. That's why every CTA today has
near-black labels.
