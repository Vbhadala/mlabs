What's in the box

Everything an agency needs on day one.

Tianjin ships the boring stuff already done so you can spend client retainers on what actually moves the needle.

Web + Mobile in one repo

Next.js App Router and Expo share the same API contracts. Build once, deploy to two surfaces without retyping a single Zod schema.

Auth, billing, email — wired

Email + magic links, Stripe checkout, transactional email — pre-integrated, with sane defaults you can ship today and tune tomorrow.

Type-safe, end to end

Zod contracts on every REST boundary. No tRPC tax, no client-server type drift. Break a schema; the build catches it.

Rebrand in one file

Every color, radius, and font flows from src/config/design.ts. Swap tokens, ship the next client. No find-and-replace.

Monorepo done right

pnpm + Turborepo, cleanly scoped @mlabs/* packages. No remote-cache lock-in. Local DX that doesn't fight you.

CI that catches regressions

Typecheck, lint, contract tests run on every PR. Mobile and web validated in parallel. Ship Friday and sleep Saturday.
